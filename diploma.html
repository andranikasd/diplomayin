English Summary

This bachelor's thesis presents the design and full-stack implementation of an intelligent chatbot platform for automated collection, storage, and AI-driven analysis of OSINT (Open-Source Intelligence) data about the Armenian business market. The system enables users to query a rich PostgreSQL database using natural language in Armenian or English, receiving statistical summaries, interactive charts, and analytical reports about Armenian companies, financial indicators, sector trends, and news articles.

The solution is built on a Node.js / Express.js backend with OpenAI GPT-4 and Anthropic Claude integrations for natural-language-to-SQL conversion, and uses Cloudflare Workers with D1 for serverless production deployment. Key components include a Redis-backed automated web scraper, Socket.io real-time communication, JWT authentication, and a layered RBAC security architecture with full audit logging.

Keywords: OSINT, business intelligence, NLP-to-SQL, LLM, Node.js, PostgreSQL, Cloudflare Workers, chatbot, Armenian companies, data analytics, RBAC.
